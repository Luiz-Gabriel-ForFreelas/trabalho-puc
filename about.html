Projeto

O projeto funciona como um pequeno cardápio on-line, que mostra na aba "home" os itens de um menu com uma opção "saiba mais..." que abre um pop-up com informações adicionais do produto desejado como nome, preço e descrição. Também é capaz de coletar dados de contato através do método GET disponível na aba "Contact", que realiza formatações para garantir que os valores sejam veridicos e válidos.

HTML e CSS

O desenvolvimento html foi responsável por extruturar o site, para que cada inoformação seja mostrada de acordo com a sequência e modelo desejado. Também responsável pelo desenvolvimento dos formulários e suas validações.

Já o CSS foi responsável pela estilização da estrutura HTML, desenvolvendo suas caracteristicas de visualização por classes e ids especificos para cada estrutura HTML.

JavaScript

O JavaScript é um tipo de programação client-side que foi utilizado para criar formatações de formulários, apresentação e manipulação de tags html e css para navegação e experiência do site, além de ser responsável de formatar os dados recebidos pelo método GET na página "Contact"
